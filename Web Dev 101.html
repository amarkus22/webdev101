Forms

action is where the data should be sent

method is what HTTP method (get/post)

name - send a name with the data. Can also connect radio buttons.

label - used for screen readers. Connected through for and id attributes.

value - adjusts what is set with the name

Always use a label. It is used for more than just screenreaders.

If you have a button in a form, the default behavior will be to submit that form, unless you specify type="button".
